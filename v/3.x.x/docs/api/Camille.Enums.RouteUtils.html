AsRegionString(PlatformRoute)

Get the slightly more human-friendly alternate name for this PlatformRoute. Specifically excludes any trailing numbers and appends extra N(orth), S(outh), E(ast), and/or W(est) suffixes to some names. Some of these are old region names which are often still used as user-facing names, e.g. on op.gg.
